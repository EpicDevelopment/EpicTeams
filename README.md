
# Epic Teams

Basic team plugin for Minecraft version 1.20.4


## Usage/Examples

```
/team invite <player>
/team chat [<message>]
/team create <team-name> <team-tag>
```


## Dependencies

 - [Vault](https://www.spigotmc.org/resources/vault.34315/)
 - [PlaceholderAPI](https://www.spigotmc.org/resources/placeholderapi.6245/)
